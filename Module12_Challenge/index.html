About This Project

This webpage is a visualization of the bacteria culture information for each volunteer. With this webpage, each volunteer is able to identify the top ten bacterial species in their belly buttons. This information can also be used by researchers if a species of bacteria is beneficial for the study.

For this project, each participant was given a unique ID to keep their identities anonymous. The user may select an ID from the dropdown menu below to filter the data on that specific ID.
With the ID selected, the participants demographic information is presented to aid in ones research. Next to the demographic information is a horizontal bar chart that displays the top ten bacterial species for the selected individual. If one hovers over the bar, the name of the bacteria associated with the OTU ID will be displayed.
The gauge meter displays the wash frequency of each participant to further visually aid in ones research.
Lastly, the bubble chart displays the bacterial data per sample for each participant where the samples are graphed by the OTU ID along the x-axis and the sample values as the y-axis. Furthermore, the size of each marker corresponds to the sample values and the color of each marker represents the OTU ID. Like the bar chart, the each marker of the hover chart will display the OTU Label corresponding to each bacterial ID.
